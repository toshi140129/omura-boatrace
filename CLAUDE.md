# 大村ボートレース 予測・自動化システム

## 環境
- OS: Windows 11
- Python 3.14.3（インストール済み）
- Node.js インストール済み
- デスクトップに omura_daily.py・omura_results.csv がある
- タスクスケジューラでログイン時に omura_daily.py が自動実行される

## ファイル構成
- omura_daily.py：毎朝自動でboatrace.jpからデータ取得しCSVに追記。GitHubにも自動push
- omura_results.csv：10R/11R/12R の着順・払戻データ蓄積
- omura_prob_analysis.html：確率表・グラフ・マーチンゲール計算ツール（ブラウザで開く）

## CSVフォーマット
日付,10R_1着,10R_2着,10R_3着,10R_払戻,11R_1着,11R_2着,11R_3着,11R_払戻,12R_1着,12R_2着,12R_3着,12R_払戻
払戻は円単位（710 = 7.1倍）

## 予測ロジック（5条件）
① 11R の1着 = 1号艇
② 10Rと11Rの3着順が完全一致しない
③ 12R の1着 = 1号艇
④ 11Rの2着 or 3着号艇が12Rの2着 or 3着に絡む
⑤ 12Rの3連単オッズが40倍未満
①が成立した日のみ発動。③④⑤がすべて成立したら的中。

## 買い目
11RのA=2着・B=3着号艇を使い
1-A-全 / 1-B-全 / 1-全-A / 1-全-B の40倍以下を全点購入
購入方式：資金配分（総賭け額 ÷ 買い目数 = 1点あたり配分額）

## マーチンゲール法（独自方式）
次回賭け金 = (累積損失 + 目標利益) ÷ 合成オッズ ※最低1万円
- 初回賭け金：10,000円
- 1節目標利益：50,000円
- 軍資金：300,000円以上

## 次のステップ（スマホWebアプリ化）
1. GitHubアカウント作成 → CSVを自動アップロード
2. Claude CodeでNext.js Webアプリを作成
3. Vercelでスマホ対応ページとして公開（無料）
4. LINE Messaging APIで条件成立時に自動通知

## 出力ルール
- 出力はすべて日本語
- 手順は最後まで一気に出す（途中で止めない）
- コマンドはそのままコピペで動く形で出す
